# Homebrew Tap

Homebrew formulae for [Lenivvenil](https://github.com/Lenivvenil) projects.

## Install

```bash
brew tap Lenivvenil/tap
brew install onenote-to-obsidian
```

## Formulae

| Formula | Description |
|---------|-------------|
| `onenote-to-obsidian` | Export OneNote notebooks to Obsidian-compatible Markdown via Microsoft Graph API |
